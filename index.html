Stage – Développement Web pour l’APEEAC (Valenciennes)

Durant mon stage au sein de l’APEEAC de Valenciennes, j’ai participé à la conception et au développement d’un site web dynamique permettant de gérer et de publier différentes informations destinées aux utilisateurs.

Ce projet m’a permis de mettre en pratique mes compétences en développement web, en gestion de base de données ainsi qu’en structuration d’une application complète.
J’ai également pu découvrir les différentes étapes d’un projet réel : analyse des besoins, conception de la base de données, développement et mise en place des fonctionnalités.

Cette expérience m’a appris à travailler sur un projet concret, à respecter des contraintes techniques et organisationnelles, ainsi qu’à collaborer avec une équipe.


Conception de la base de données

Avant de commencer le développement du site, la base de données a été conçue à l’aide d’un MCD (Modèle Conceptuel de Données).

Le MCD permet de structurer les informations nécessaires au fonctionnement du site en identifiant les différentes entités, leurs attributs ainsi que les relations entre elles.
Cette étape est essentielle pour organiser correctement les données et faciliter le développement de l’application.


Interface utilisateur

Le site propose une interface simple et accessible permettant aux utilisateurs de consulter les informations disponibles et d’accéder aux différentes fonctionnalités du service.
Voici un aperçu de la page d’accueil du site du point de vue d’un utilisateur.


Fonctionnalités dynamiques

Plusieurs fonctionnalités dynamiques ont été développées afin de permettre l’affichage et la gestion des données directement depuis la base de données.


Système de petites annonces

Le site intègre également un système permettant aux utilisateurs de publier et consulter des petites annonces via un formulaire dédié.


Interface d’administration

Une interface d’administration a été développée afin de permettre aux administrateurs de gérer le contenu du site ainsi que les différentes données présentes dans la base de données.
Cette interface est accessible uniquement aux administrateurs.
Elle leur permet d’ajouter, de modifier ou de supprimer des visuels pour l'utilisateurs ainsi que de gérer les différentes sections du site.
Par exemple, les administrateurs peuvent ajouter des archives et des evements, gérer les petites annonces, ou encore changer les couleur du sites .
